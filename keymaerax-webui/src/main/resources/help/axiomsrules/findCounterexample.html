Find counterexample searches values that satisfy all the assumptions but none of the conclusions. Inspecting the counterexample values may hint at missing assumptions (e.g., unexpected negative values) or point to missing conditions that link variables (e.g., high speed with no safety distance left).

See also

  • Tools → QE for quantifier elimination if no counterexample is found.
  • Tools → Find assumptions to get hints on how to strengthen the assumptions to make the sequent provable.
