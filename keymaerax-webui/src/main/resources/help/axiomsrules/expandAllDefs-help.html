expandAllDefs expands all definitions of function symbols, predicate symbols, and program symbols according to the Definitions block of the model file.

For example, occurrences of dist(a,b+1) are expanded to a^2+(b+1)^2 etc. given these definitions: 

Definitions
  Real dist(Real x, Real y) = x^2+y^2;
  Bool sep(Real x, Real y) <-> x^2+y^2 >= 9;
  HP ctrl ::= {x:=x+1;};
End.
